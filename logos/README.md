# TheProfesseer — Logo Kit

Four assets. Pick by **what you're placing** (full logo vs. icon) and **what's
behind it** (light vs. dark background). All have transparent backgrounds.

## The four

| | Crimson (on light) | White (on dark) |
|---|---|---|
| **Lockup** (logo + text) | `lockup/professeer-logo-crimson` | `lockup/professeer-logo-white` |
| **Mark** (logo only) | `mark/professeer-mark-crimson` | `mark/professeer-mark-white` |

- **Lockup** = owl + "The Professeer" wordmark. Use anywhere with room for the
  name (documents, decks, site header, email signature, letterhead).
- **Mark** = owl on its own. Use where the wordmark would be illegible (favicon,
  avatar, watermark, tight spaces).
- **Crimson** (`#7B0925`) goes on light backgrounds; **white** goes on dark or
  crimson backgrounds. Never crimson-on-dark or white-on-light — contrast fails.

## File formats

Each asset comes in two formats:

- **SVG** — vector master. Use everywhere you can (web, slides, LaTeX, print);
  scales with no quality loss.
- **PNG** — transparent raster (~1200px tall). Use only where SVG isn't accepted.

## Don'ts

Don't recolor, stretch, rotate, add shadows, or crowd the mark. Keep clear space
around it equal to the height of the owl's head.
